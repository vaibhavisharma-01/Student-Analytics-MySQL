# Student Analytics (SQL Project)

This mini project demonstrates use of SQL in MySQL database to manage and analyze student academic data.

## 📁 Files Included
- students.csv
- courses.csv
- marks.csv
- create_tables.sql
- insert_data.sql
- analysis_queries.sql

## 🚀 How to run
1. Run create_tables.sql in MySQL
2. Load CSVs using insert_data.sql
3. Run analysis_queries.sql to get outputs
